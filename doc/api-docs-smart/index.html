      核心高并发接口，采用异步削峰架构。
      前端需轮询查询结果。



      业务流程：
      <ul>
      <li>1. 本地缓存检查库存（阻挡90%无效请求）</li>
      <li>2. Redis Lua脚本原子扣减库存</li>
      <li>3. 发送MQ消息异步创建订单</li>
      <li>4. 立即返回排队状态</li>
      </ul>
