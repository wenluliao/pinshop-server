2.1.执行秒杀下单


核心高并发接口，采用异步削峰架构。
前端需轮询查询结果。



业务流程：
<ul>
<li>1. 本地缓存检查库存（阻挡90%无效请求）</li>

<li>3. 发送MQ消息异步创建订单</li>
<li>4. 立即返回排队状态</li>
</ul>

URL: http://localhost:8080/api/v1/trade/seckill

Type: POST

Author: FlashBuy Team

Content-Type: application/json

Description: 1. 必须先登录；2. 接口含防刷限流；3. 单次限购1件

Request-headers:

HeaderTypeDescriptionRequiredSince

Authorization

string

认证Token (Bearer {jwt_token})

false

-

ParameterTypeDescriptionRequiredSince

int64

false

int32

false

-

userId

int64

用户ID（必填，从登录Token中获取）

false

-

  "eventId": 0,
